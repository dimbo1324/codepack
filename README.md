# Project Exporter Desktop

Десктопное приложение для Windows, которое создаёт безопасный, структурированный снимок любого проекта — для передачи ИИ-инструментам, другому разработчику или для архивного хранения.

**Версия:** 5.1.0 → в разработке  
**Платформа:** Windows 10 / 11  
**Стек:** Python 3.11+ · PySide6 · PyInstaller · Inno Setup

---

## Что это такое

Project Exporter Desktop — инструмент разработчика, который решает одну задачу хорошо:

> Выбрать проект → увидеть, что попадёт в экспорт → одним нажатием создать безопасный, понятный архив с отчётами.

Приложение автоматически определяет стек проекта (Node.js, Python, Go, Java, Rust и другие), применяет правильные правила исключения, убирает секреты и генерирует более 30 отчётов о структуре, зависимостях, безопасности и архитектуре.

---

## Основные возможности

- **Авто-определение стека** — распознаёт язык и фреймворк, подставляет готовые ignore-шаблоны
- **Интерактивное дерево файлов** — видно, что войдёт в экспорт, до запуска; можно редактировать
- **Счётчик токенов** — показывает, сколько токенов займёт дамп в Claude, GPT-4, Gemini
- **Быстрый clipboard** — скопировать весь текстовый дамп в буфер за одно нажатие
- **Контекст разработчика** — поле для заметок, которые попадут в AI-контекст первыми
- **Дифференциальный экспорт** — только файлы, изменённые с последнего экспорта или git-ссылки
- **Безопасность** — три режима (Строгий / Сбалансированный / Полный), сканирование секретов, редактирование
- **30+ отчётов** — структура, Git, AI-контекст, зависимости, граф импортов, health score, HTML-дашборд
- **Русский интерфейс** — полная локализация UI, диалогов и отчётов
- **История экспортов** — сохранение, сравнение снапшотов, быстрый повторный запуск

## Запуск из исходников

```bat
py -3.11 -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Тесты

```bat
python -m pytest
```

## Сборка EXE

```bat
python tools\build_exe.py
```

Результат: `dist\ProjectExporterDesktop.exe`

## Сборка установщика

Требуется [Inno Setup](https://jrsoftware.org/isinfo.php). После установки:

```bat
tools\build_setup.bat
```

Скрипт установщика: [`build\installer\ProjectExporterDesktop.iss`](build/installer/ProjectExporterDesktop.iss)

## Структура проекта

```
src/project_exporter_desktop/
├── gui/              # Интерфейс (5+ страниц, воркеры, стили)
├── services/         # Бизнес-логика (план, копирование, архивация, политика)

```
